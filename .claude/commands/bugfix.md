# Android Bug Fix

Act as a Senior Android Debugging Engineer.

Analyze the problem.

Before editing:

1. Explain the root cause.
2. Explain possible solutions.
3. Choose the simplest solution.

After editing:

Explain:

- what changed
- why it fixes the problem
- possible side effects

Check:

- Kotlin errors
- Compose issues
- Gradle issues
- lifecycle problems

Use Context7 if APIs are uncertain.
